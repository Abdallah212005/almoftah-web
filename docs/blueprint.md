# **App Name**: Estate Connect

## Core Features:

- Unit Listing: Admin uploads real estate units (apartments, villas, offices, lands) with details like price (in EGP), photos, and descriptions.
- Lead Management: Admin adds and manages leads (potential customers) and brokers (external brokers).
- Real-time Chat: Built-in chat system for admins to respond to customer inquiries.
- Unit Filtering: Users can filter units by category (apartment, villa, etc.), city, and governorate.
- Unit Request: Users can request specific units and initiate chats with admins about them.
- Admin Role Management: Super admin (abdallah amr) can create, hide/show, and assign tasks to admin users.
- AI-Powered Property Suggestions: Leverage generative AI as a tool to analyze user preferences and suggest relevant properties.

## Style Guidelines:

- Primary color: Soft gray (#E0E0E0) for a clean, modern look.
- Background color: Off-white (#FAFAFA) to provide a neutral backdrop.
- Accent color: Dark blue (#2E3192) to highlight key interactive elements, maintaining a professional and trustworthy feel.
- Body and headline font: 'Inter', a sans-serif font with a modern, machined, neutral look that provides excellent readability for both headlines and body text. 
- Clean, minimalist icons for unit categories, location filters, and user actions.
- Logo.png placed at the top left corner. Clean and structured layout with clear separation of sections.
- Subtle transitions and animations to enhance user experience (e.g., loading animations, hover effects on unit cards).
